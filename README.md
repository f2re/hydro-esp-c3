# 🌿 Hydroponics Tower ESP32-C3

Профессиональная прошивка для автоматизации гидропонной башни на базе **ESP32-C3 Super Mini** с OLED-дисплеем и интеллектуальным управлением.

---

## 🎯 Основная цель
Обеспечить стабильный полив растений по научно обоснованному расписанию, используя минималистичный, но информативный интерфейс (OLED + Serial ANSI) и защищенное управление реле.

---

## ✨ Ключевые особенности
- 🖥️ **OLED Ротатор (72x40):** Автоматическая смена информационных экранов:
  - **Часы:** Крупное время, секунды и статус WiFi (RSSI).
  - **Таймер:** Обратный отсчет до ближайшего полива.
  - **Расписание:** Список ближайших 4 слотов с выделением активного.
  - **Полив:** Приоритетный экран с анимацией и прогресс-баром.
- 🕐 **Точное время:** Синхронизация с NTP (UTC+3 по умолчанию).
- 💧 **Профессиональное расписание:** 9-слотный цикл (день/ночь), оптимизированный для аэропоники.
- 🛡️ **Стабильность железа:**
  - Реле в режиме `OUTPUT_OPEN_DRAIN` (GPIO4).
  - Отключение Brown-out детектора для предотвращения перезагрузок при пуске помпы.
- 🔘 **Интерактивность:**
  - **BOOT кнопка:** Переключение страниц OLED или принудительная остановка полива.
  - **LED индикатор:** Мигает во время работы помпы.
- 📡 **WiFi Manager:** Авто-подключение и вывод уровня сигнала в dBm.

## 🛠️ Настройка и запуск

### 1. WiFi учетные данные
Задайте переменные окружения в терминале (безопасный способ):
```bash
export WIFI_SSID="My_WiFi"
export WIFI_PASSWORD="My_Password"
```

### 2. Сборка и прошивка
```bash
pio run --target upload
```

### 3. Мониторинг
```bash
pio device monitor
```

## 🖥️ Интерфейсы

### OLED Дисплей (Ротация 5 сек)
1. **Clock:** `22:15:30` | `RSSI: -65dB`
2. **Next:** `Next 08:00` | `in 09h 45m`
3. **Schedule:** Список слотов с текущим прогрессом.
4. **Watering:** `WATERING` | `45s left` | `[████░░]`

### Serial ANSI Дашборд
Очищает экран и рисует таблицу статуса каждые 5 секунд (цветной вывод).

## 🔌 Схема подключения (ESP32-C3 Super Mini)
| Компонент | Пин ESP32 | Примечание |
|-----------|-----------|------------|
| **Relay IN** | GPIO 4 | Режим Open Drain |
| **OLED SDA** | GPIO 5 | I2C (через Wire) |
| **OLED SCL** | GPIO 6 | I2C (через Wire) |
| **LED** | GPIO 8 | Встроенный (Blink при поливе) |
| **Button** | GPIO 9 | BOOT кнопка (Page switch) |
