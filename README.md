# 🌿 Hydroponics Tower ESP32-C3

Прошивка для автоматизации полива гидропонной башни на базе контроллера **ESP32-C3 Super Mini**.

---

## 🎯 Основная цель
Обеспечить надежный и точный полив растений по расписанию, используя синхронизацию времени через интернет (NTP) и предоставляя удобный визуальный мониторинг через последовательный порт (Serial) без бесконечного скролла.

---

## ✨ Ключевые особенности
- 🕐 **Точное время:** Синхронизация с NTP-серверами (pool.ntp.org).
- 💧 **Умный полив:** Настраиваемое расписание с точностью до минуты и секунды.
- 🖥️ **Live Dashboard:** Динамический интерфейс в терминале через ANSI-коды (обновление на месте).
- 📡 **WiFi Manager:** Автоматическое переподключение при потере связи.
- ⚡ **Non-blocking:** Использование `millis()` вместо `delay()` для плавной работы всех систем.
- 🔐 **Безопасность:** WiFi учетные данные передаются через переменные окружения, а не хранятся в коде.

## 🛠️ Настройка и запуск

### 1. Подготовка окружения
Установите [PlatformIO](https://platformio.org/) (рекомендуется плагин для VS Code).

### 2. Конфигурация WiFi
Не вписывайте пароль в код! В терминале перед прошивкой выполните:
```bash
export WIFI_SSID="Ваше_Название_Сети"
export WIFI_PASSWORD="Ваш_Пароль"
```

### 3. Настройка расписания
Отредактируйте `src/config.h` для изменения времени полива:
```cpp
static const WateringSlot WATERING_SCHEDULE[] = {
    {8,  0, 300},   // 08:00 — 5 минут
    {12, 0, 300},   // 12:00 — 5 минут
    {22, 0, 180},   // 22:00 — 3 минуты
};
```

### 4. Прошивка
Подключите ESP32-C3 к USB и выполните:
```bash
pio run --target upload
```

### 5. Мониторинг
Откройте монитор порта, чтобы увидеть дашборд:
```bash
pio device monitor
```

## 🔌 Схема подключения (default)
- **Relay (Pump):** GPIO 5 (HW-307 или аналогичный с активным LOW)
- **Power:** USB-C или 5V на соответствующие пины ESP32-C3.
